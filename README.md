## &nbsp;Submission Standard

<br/>

> [!IMPORTANT]
> The final repository will contain only reviewer-relevant material:

<br/>

<table>
  <tr>
    <td>

&nbsp;&nbsp;🔹&ensp;Source code required to run the solution
<br/>
&nbsp;&nbsp;🔹&ensp;A precise problem and solution narrative
<br/>
&nbsp;&nbsp;🔹&ensp;Reproducible setup and demo instructions
<br/>
&nbsp;&nbsp;🔹&ensp;Architecture and data-flow documentation
<br/>
&nbsp;&nbsp;🔹&ensp;Validation, error handling, and test evidence
<br/>
&nbsp;&nbsp;🔹&ensp;Meaningful commit history from both team members
<br/>
&nbsp;&nbsp;🔹&ensp;Screenshots or a short demo link when available

  </td>
  </tr>
